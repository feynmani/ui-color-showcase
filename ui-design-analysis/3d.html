代码实现

CSS 3D代码实现

.d3-element {
    transform-style: preserve-3d;
    perspective: 1000px;
    transition: transform 0.5s ease;
}

.d3-element:hover {
    transform: rotateY(15deg) rotateX(5deg);
}

.d3-card {
    width: 200px;
    height: 250px;
    position: relative;
    transform-style: preserve-3d;
    transform: translateZ(0);
}

.d3-card::before,
.d3-card::after {
    content: '';
    position: absolute;
    width: 100%;
    height: 100%;
    background: inherit;
    border-radius: 1rem;
    transform: translateZ(-20px);
    opacity: 0.8;
}

.d3-card::after {
    transform: translateZ(-40px);
    opacity: 0.6;
}

HTML结构

<div class="d3-container">
    <div class="d3-element">
        <div class="d3-card">
            <h3>3D内容</h3>
            <p>现代3D设计</p>
        </div>
    </div>
</div>

最佳实践：

  • 使用transform-style: preserve-3d创建真正的3D效果
  • 添加perspective属性增强立体感
  • 优化3D性能，避免过度复杂的变换
  • 确保良好的对比度和可读性
